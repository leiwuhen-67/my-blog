vue项目安装依赖报错

好久没有手动从头开始搭建一个新项目了，最近从头搭建一个项目，安装 vant 时报如下错误

从报错信息看，是因为 eslint-plugin-vue 与 @vue/eslint-config-standard 版本不兼容导致的。

这里有两种解决办法：

1、通过 --force 或 --legacy-peer-deps 命令来强制安装，能安装成功，但是可能会导致未知错误。

npm install vant -S --force  
// 或 npm install vant -S --legacy-peer-deps
1
2

2、按报错的信息来看，可以降级 eslint-plugin-vue 的版本来解决。

在 package.json 中可以看到 @vue/eslint-config-standard 的版本是 6.1.0，而 eslint-plugin-vue 的版本是 8.0.3，从报错信息看，与之对应的 eslint-plugin-vue 版本是 7.0.0，因此可以将 eslint-plugin-vue 降级到 7.0 版本

npm install eslint-plugin-vue@7.0.0 -D
1

现在重新安装 vant 版本发现是可以解决问题的
