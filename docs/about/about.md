# 介绍
<hr />
你们好，我，一个搬砖人。生命不息，折腾不止，从最初的<a href="https://www.cnblogs.com/jf-67/" target="_blank">博客园</a>到后来的用<a href="https://leiwuhen-67.github.io/" target="_blank">Hexo + GitHub Pages</a>搭建博客，再到现在的vuepress + GitHub Pages搭建博客，虽说博客不常写，但是该有的咱还是得有。


在我看来，写博客有诸多好处：


1、整理自己的知识体系。在记录的过程中帮助我们理清头绪，理清思路，加身理解与记忆，就算过一段时间忘了，再次遇到同样的问题，可以看一遍文章，能少走许多弯路。


2、记录。记录一个新奇的技术点，一个bug的解决，亦或是一段生活经历，在以后的某个时间段再回过头来看会有别样的感触。
